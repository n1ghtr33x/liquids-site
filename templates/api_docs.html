Документация API

GET /api/liquids

Возвращает JSON-список всех доступных жидкостей.

Пример ответа:

            
[
  {
    "id": 1,
    "name": "Название жидкости 1",
    "price": 150.0
  },
  {
    "id": 2,
    "name": "Другое название",
    "price": 200.5
  }
]

GET /api/analytics

Возвращает JSON-объект с количеством голосов для каждой жидкости.

Пример ответа:

            
{
  "Название жидкости 1": 12,
  "Другое название": 5
}
